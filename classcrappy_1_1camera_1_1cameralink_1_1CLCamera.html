Using the clModule, will open a cameraLink camera.

If a config file is specified, it will be used to configure the camera If not set, it will be asked, unless set to False (or 0) Else, you must at least provide the camera type (eg: "FullAreaGray8") Using a config file is recommended over changing all settings manually
