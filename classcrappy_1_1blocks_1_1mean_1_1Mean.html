Detailed Description

Can take multiple inputs, makes an average and send the result every delay s.

Parameters
tlabelrefers to the label that carries the time (default='t(s)')
out_labelsif given, only these labels and the time will be kept and returned. If none (default), all of them will be kept.
freqThe block will loop at this freq. see MasterBlock.freq
