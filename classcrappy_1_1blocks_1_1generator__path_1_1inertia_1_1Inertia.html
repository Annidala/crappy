Detailed Description

Used to lower/higher the output command by integrating an input over time.

Let f(t) be the input signal, v(t) the value of the output, m the inertia and t0 the beginning of this path. K is a choosen constant. The output value for this path will be: v(t) = v(t0) - K*[I(t0->t)f(t)dt]/m

inertia (float): This is the virtual inertia of the process. The higher it is, the slower the (in/de)crease will be. In the above formula, it is the value of m flabel (str): The name of the label of the value to integrate tlabel (str), default is 't(s)': The name of the label of time for the integration

Note
The data received by flabel and tlabel must correspond. In other word, there must be exactly the same number of values received by these two labels at any instant (ie they must come from the same parent block). const (float), default is 30/pi: The value of K in the formula above. The default value is meant to send rpm with inertia in kgmÂ² and torque in Nm. If sending rad/s, use const=1 condition (str, see Path for more info): Condition to meet to end this path
