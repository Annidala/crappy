modestr. Available modes at the moment :
singleOutput is (measured_value * gain) + offset (~2kHz max.)
thermocoupleOutput is a temperature in degree celsius.
streamerOutput is (measured_value * gain) + offset (100 kSamples max.)
channelsint, str or list of int or str, default = 0 (AIN0)
gainfloat or list of float, default = 1
offsetfloat, default = 0
chan_rangeint or float, default = 10. Can be 10, 1, 0.1 or 0.01, depending on the voltage range to measure. Put the absolute maximum of your expected values. The higher the range, the fastest the acquisition rate is. resolution: int, resolution index for each channel.
T71 to 8, T7-PRO : 1 to 12. If 0 is specified, will be 8 (20 bits) Check https://labjack.com/support/datasheets/t7/appendix-a-3-1 for more
information.
scan_rate_per_channelSTREAMER MODE ONLY : int, defines how many scans to perform on each channel during streaming.
