Detailed Description

Block to make a visual representation of data.

Parameters
imageThe only mandatory argument. This image will be the background for the Canvas.
drawA list of dict defining what to draw. Each dict must contain a 'type' key that contains the name of the element, Drawing will then create the corresponding class with all the other keys as argument.
